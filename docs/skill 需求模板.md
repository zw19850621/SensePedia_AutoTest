## 8. 配置要求

需要在配置文件中配置的内容：

### config/scenarios.yaml

```yaml
scenarios:
  [scenario_name]:
    name: "场景名称"
    document_upload:
      base_path: "文档目录路径"
      file_types: ["pdf", "docx", "md", "txt"]
      language: "zh-cn"
      visibility: "private"
```

### config/auth.yaml

```yaml
auth:
  credentials:
    username: "用户名"
    password: "密码"
```
